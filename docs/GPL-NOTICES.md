# GPL Notices

`warden-free` is intended to be distributed under GNU GPL v2.

This repository contains:

- the Qt6 GUI for the Free edition
- ClamAV scan integration through `libclamav`
- optional definition update integration through `libfreshclam`
- the `warden-clamav-bridge` companion executable

When a Pro release bundles `warden-clamav-bridge`, the matching Pro release should also reference the exact `warden-free` source tag used to produce that binary.
